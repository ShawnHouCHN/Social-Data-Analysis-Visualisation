Assignment 2B: Visualizing Geodata

Below is an interactive KMeans clustering for prostitution crimes in San Fransisco can be seen. The data used is the total amount of prostitution crimes in San Francisco from 2003 to present time (around 16,000 data points). In the visualization one can alter the K-value between 2 and 6, and see the resulting clustering. The data file used was generated and exported to JSON format using Python, with a file size of 676 KB.

In addition to visualising the clusterings, their stability is also provided. Stability analysis is used to estimate how well a given clustering fits to the data. By doing N random samplings from the dataset and examining the centroid locations for each K after clustering, one can see how “stable” the clusterings are as a function of K. If the cluster centers tend to land in the same place (determined by the closest grid point on a square grid) for different Ns (samplings), the clustering is said to be stable. This is measured by calculating the intra cosine similarities of each centroid location (that is if the cluster centers are often assigned to the same grid point, there will be a high cosine similarity for that clustering).

In the case of prostitution in San Fransisco, the clustering with K = 3 gave the stability. The stability analysis can be seen in the iPython notebook provided above (and here).

Notice the nodes color rendering and transiting do not take place at exact same time. The animation would last for a second and nodes are rendered gradually.
